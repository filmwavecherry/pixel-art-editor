# pixel-art-editor

A browser-based tool for turning photos and videos into pixel art.

## Features

- Upload an image or video and pixelate it in real time
- Nearest-neighbor pixelation with adjustable pixel size (2–128px)
- Color correction: brightness, contrast, saturation, temperature, tint
- Color quantization with adaptive palette or named palettes (Game Boy, Pico-8, CGA, NES)
- Zoom and pan the canvas
- Video: live pixelated preview, trim controls, framerate control
- Export as PNG (images) or MP4 (video, via WebCodecs)

## Stack

Plain HTML/CSS/JS — no frameworks, no build step. Open `index.html` in a browser.

## Status

V1 (image) and V2 (video) complete. V3 (procedural animation effects) in progress.
